# My Portfolio
My portfolio is a simple website for potential freelance clients and for displaying my work. 

**Link to project:** [Blakelarson.dev](https://blakelarson.dev/)

![The landing page for Blake's Website.](https://blakelarson.dev/images/portfolio-cover.jpg)

## How It's Made:

**Tech used:** HTML, CSS, JavaScript, TailwindCSS

At the time I made this, TailwindCSS was fairly new to me. I had used Bootstrap or vanilla CSS before and was looking for another framework to learn. I recreated my portfolio using TailwindCSS to see what it is capable of. I really enjoyed using TwailwindCSS and I'm looking forward to using it on other projects. It was easy to bring in the classes they provide and I also had the flexibility to create my own styles and website design. This can be hard to achieve with other frameworks like Bootstrap.

I also added a few small JavaScript functions for the Scroll to Top button and the hamburger menu for mobile. 

## Optimizations

None yet, but I will continue to work on this over time and will add any optimizations I make.

## Lessons Learned:

This project was all about learning how to use TailwindCSS. It was a great place to expirement in a low stakes environment because it was my site! I also learned how TailwindCSS handles animations and how I can use simple JavaScript with TailwindCSS to quickly make interactive elements.
