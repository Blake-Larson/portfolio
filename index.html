About

I am a man of many hobbies...

I love finding new things to dive into. Throughout the years two fields have always stuck with me: I love creating music and I love tech and the potential good it can bring to our world as a tool.

I decided to get a degree in music education and began teaching private lessons out of college but the pandemic brutally shifted how I had to operate and I ran into many technological barriers to my teaching. I learned JavaScript to build better tools for myself and my students. This reinvigorated my love of tech through coding and pushed me to spend more time on my projects.

I have since transitioned to freelancing as a web developer with a focus on MERN stack technologies and love to build websites and web apps for clients. I am passionate about stewarding the environment and creating positive change for people.

NDA-Protected React App

for Austin Arey and Logan Wroolie

Tech Used: React, WebSockets, TailwindCSS

This project is protected by an NDA, therefore, this description will focus on the tech and the process used to build the application.

I was hired to build a Front-End React application in conjunction with a Django backend built by Austin Arey. For a little over a month, we collaborated and succeeded in creating an app that takes in user form data, determines settings from that data, and displays real-time game data via WebSockets. We also made a fully functional in-game menu that would stop, pause, resume, or restart the real-time stream.

Throughout the process, I learned how to integrate my Front-End application with Django and improved my communication skills by collaborating with Austin on JSON object architecture.I implemented Routes for this project with React-router-dom and I used TailwindCSS as my CSS framework. Additionally, I added a custom CSS animation upon request from Austin and Logan. Overall, this project went smoothly and I enjoyed my time working with Austin to build this application.
